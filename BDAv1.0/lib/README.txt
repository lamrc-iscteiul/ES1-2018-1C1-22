Para utilizar as libraries:

restfb-2.11.0.jar:
	-Go to Project>Properties>Java Build Path>Libraries>Add External JAR File
	
restfb-2.11.0-javadoc.zip:
	-Unpack the .zip file
	-Go to Project>Properties>Java Build Path>Libraries
	-Expand the restfb library
	-Click on "javadoc location"
	-Click "Edit" on the menu on the right
	-Select the unpacked folder
	
selenium-java-3.14.0:
	-Unpack the .zip file
	-Go to Project>Properties>Java Build Path>Libraries>Add External JAR File
	-Add all .jar files in the folder, including the ones on selenium-java-3.14.0/lib
